# Flask Lead Collector

A simple Flask app to collect leads (names and emails) from users through a web form. Perfect for small businesses or marketing campaigns to gather contacts efficiently.

## Features

- Collect user names and emails via a web form  
- Save collected leads in a CSV file (`leads.csv`)  
- Lightweight Flask application with minimal setup  
- Can run locally or on a cloud server (e.g., AWS EC2)  
- Ready for integration with email campaigns or notifications

## Demo

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/Husmankol/flask-lead-collector.git
   cd flask-lead-collector
2. Go into folder
3. Create virtual environment
   python -m venv venv
4. Activate virtual environment
   venv/bin/activate
5. Install requirements
   pip install -r requirements
6. Start the app
   python happ.py


## Usage

1. Collect names and emails from users.

2. Leads are saved in leads.csv.

3. Can be integrated with email campaigns, WhatsApp, or Telegram for follow-ups.
